# 🥃 Agua o Tequila Online

## Setup rápido

### 1. Backend
```bash
cd backend
npm install
npm run dev
```
Corre en `http://localhost:4000`

### 2. Frontend (otra terminal)
```bash
cd frontend
npm install
npm run dev
```
Corre en `http://localhost:3000`

### 3. IA (opcional)
Para habilitar afirmaciones con IA, crea un archivo `backend/.env`:
```
ANTHROPIC_API_KEY=sk-ant-...
```
Sin API key funciona con el pool local de 75 afirmaciones.

## Probar
1. Abre `http://localhost:3000` en tu navegador
2. Crea una sala
3. Abre otra pestaña/incógnito y únete con el código
4. ¡Juega!
